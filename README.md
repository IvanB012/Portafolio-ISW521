# Portafolio - ISE521 (Desarrollo Web)

Bienvenido a mi portafolio de evidencias de trabajo en clase para el curso ISE521. En este repositorio se documenta el progreso y las actividades realizadas a lo largo del curso, evidenciando la evolución de los conocimientos adquiridos.

## 🚀 Evolución y Actividades

### 📘 Clase 1: Fundamentos de la Web y Redes
En esta primera unidad nos introducimos a los conceptos teóricos fundamentales de la web y la arquitectura de red:
- **Actividad 1**: Diferenciación entre Internet y la World Wide Web (WWW), así como el rol de organismos estandarizadores como W3C y IETF.
- **Actividad 2**: Análisis de casos de uso y diferencias entre Internet, Intranet y Extranet, junto con estrategias de aislamiento lógico y seguridad.
- **Actividad 3**: Desglose y comprensión de la estructura de las URLs (protocolo, host, puerto, path, parámetros, fragmentos) y las mejores prácticas para su diseño.
- **Actividad 4**: Estudio de la jerarquía de dominios (TLD, SLD, Subdominios), su administración (ICANN, Verisign, NIC Costa Rica) y los tipos de registros DNS (A y CNAME).

*(Para leer las respuestas completas, consultar: [`Ejercicios/Clase 1/actividades.md`](./Ejercicios/Clase%201/actividades.md))*
